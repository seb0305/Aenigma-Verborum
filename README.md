# Aenigma Verborum (Latin-German Learning Project)

## About

This project is a web-based Latin vocabulary game for students that combines a personal vocab book, adaptive quizzes, and collectible trading cards. Students manually enter Latin words with German translations, then the app generates quiz rounds that focus on the words they struggle with most.
